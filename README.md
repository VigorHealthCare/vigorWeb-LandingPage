# VigorCare Landing Page

A premium, high-fidelity landing page for VigorCare, a remote healthcare platform designed for families sponsoring healthcare for loved ones in Nigeria.

## 🚀 Overview

VigorCare bridges the accountability gap in healthcare sponsorship. This landing page is built to capture leads via a waitlist and showcase the platform's core value propositions, including EMR transparency and verified provider networks.

## ✨ Core Features

- **High-Fidelity UI:** Modern, responsive design with glassmorphic elements and professional branding.
- **Single-Page Architecture:** Narrative flow covering the problem space, features, network, and pilots.
- **Lead Capture:** Integrated Google Form for waitlist management.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewports.
- **Branded Assets:** Official logos and high-quality imagery localized in the `assets/` directory.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom design system using CSS Variables and Flexbox/Grid.
- **Vanilla JavaScript:** Smooth scrolling, sticky navigation, and mobile menu interactions.

## 📂 Project Structure

- `index.html`: Main page content and structure.
- `index.css`: Stylesheet with responsive design tokens.
- `main.js`: Interaction logic and navigation behavior.
- `assets/`: Directory containing logos, background images, and app badges.

## 🏁 Local Development

To run this project locally:

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd vigorLandingPage
   ```

3. Open `index.html` in your browser or use a local development server:

   ```bash
   # Using Python
   python -m http.server 8000
   ```
